📘 Projeto: Tabuadas do 1 ao 10
📝 Descrição

Este projeto é uma aplicação web simples que gera e exibe as tabuadas do 1 ao 10 de forma organizada e visualmente agradável.

Ao clicar no botão "Gerar Tabuadas", o sistema exibe as operações de multiplicação de cada número (de 1 a 10), com seus respectivos resultados, organizadas em cards individuais.

🎯 Objetivo da Atividade

Praticar HTML para estruturação da página.

Utilizar CSS para estilização e organização visual.

Aplicar JavaScript para geração dinâmica de conteúdo.

Trabalhar com:

Laços de repetição (for)

Manipulação do DOM

Eventos (onclick)

Criação dinâmica de elementos

🖥️ Funcionalidades

✔️ Botão para gerar as tabuadas
✔️ Exibição das tabuadas do 1 ao 10
✔️ Multiplicações de 1 até 10 para cada número
✔️ Layout organizado em cards
✔️ Interface moderna com cores e sombra

🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

📂 Estrutura do Projeto
📁 projeto-tabuadas
 ├── index.html

🚀 Como Executar o Projeto

Baixe ou clone o repositório.

Abra o arquivo index.html em seu navegador.

Clique no botão "Gerar Tabuadas".

Visualize as tabuadas sendo exibidas na tela.

🧠 Lógica Utilizada

O JavaScript utiliza:

Um laço externo para percorrer os números de 1 a 10.

Um laço interno para gerar as multiplicações de 1 a 10.

Manipulação do DOM para criar dinamicamente os elementos HTML.

Inserção de conteúdo usando innerHTML ou appendChild.

🎨 Layout

Fundo azul escuro

Títulos em amarelo

Cards com sombra

Organização responsiva
